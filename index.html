What exactly is Exponent API?


Exponent APITM is a library of independent X-Tensions (Dynamic Link Libraries) designed exclusively to run inside of X-Ways Forensics (XWF). Each X-Tension in the library is a standalone DLL that is loaded (executed) from within the XWF software. An open Case File is required for most of the X-Tensions. Exponent APITM is designed to extend and enhance the functionality of XWF by supporting new artifacts and in some cases improving the manner in which evidence is displayed, managed or reported.

 

How is Exponent API licensed?


Use of the Exponent APITM is managed via a non-perpetual subscription-based license.

Each license is tethered to a specific X-Ways Forensics dongle (the nLicID found in the About window). A single Exponent APITM license file can contain licensing information for several X-Ways Forensics dongles. This makes it easy to support a forensic lab with multiple X-Ways Forensics dongles. Simply copy the Exponent APITM licence file to the X-Ways Forensics installation directory on each workstation. No extra dongles to fuss over.

Exponent APITM is licensed as a 1-year subscription. If a subscription is allowed to expire, then the software no longer functions. Please note that multiple year subscriptions are available.
